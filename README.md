Slyce iOS SDK
==============

The Slyce iOS SDK makes it easy to add visual search capabilities to mobile apps.

### [Documentation](https://docs.slyce.it)

### Carthage Support

To leverage release 5.1.0 of the SlyceSDK, add the following line to your Cartfile.
```
binary "https://raw.githubusercontent.com/Slyce-Inc/Slyce-iOS-SDK/master/Carthage/SlyceSDK.json" == 5.1.0
```
